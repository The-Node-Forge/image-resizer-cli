### Prerequisites

- Node.js version 14.x or later
- npm or yarn

### Installation Steps

```sh
npm install -g @the-node-forge/image-resizer-cli
```

or using Yarn:

```sh
yarn global add @the-node-forge/image-resizer-cli
```

For more details, see [USAGE.md](USAGE.md).
